# 项目范围

## 产品定位

Interview Training Workbench 是一个 local-first 的面经训练台，用来把零散面经文章转成可练习、可评分、可复盘的题目闭环。

核心训练对象是“问题”，不是“文章”。文章只是输入来源，最终沉淀物是问题、回答记录、评分反馈和正式卡片。

## Phase A 北极星

Phase A 只服务一条红线路径：

```text
导入 / 抓取文章 -> 抽取问题 -> 选题作答 -> 评分反馈 -> 重答对比 -> 保存卡片
```

## 产品原则

- 本地优先：数据落在 `data/`，不依赖账号、数据库和云同步。
- 闭环优先：先保证能练、能评、能重答、能沉淀。
- 问题优先：面经文章不直接成为训练对象。
- 结构化反馈优先：评分必须能指出技术、表达、工程思维短板。
- 轻后端：Phase A 使用 Node HTTP server + JSON / JSONL，避免过早工程化。
- LLM 可选：有 key 时自动抽题/评分；无 key 时仍可手动导入和粘贴评分 JSON。

## 明确非目标

- 多用户、登录、云同步。
- SQL / ORM / 后台管理系统。
- 多信息源平台化。
- 用户画像、弱项推荐、Review Mode、Learn Mode。
- 复杂调度、间隔重复、智能推荐算法。

这些能力进入 Phase B / C，而不是继续塞进 Phase A。
